Время полёта:
  • К собственному полю обломков:
    (10 + (35000 / процент* корень(5000 / скорость))) / (24 * 60 * 60)

  • В системе:
    (10 + (35000 / процент* корень((1000000 + удалённость планеты* 5000) / скорость))) / (24 * 60 * 60)

  • В галактике:
    (10 + (35000 / процент* корень((2700000 + удалённость системы* 95000) / скорость))) / (24 * 60 * 60)

  • Между галактиками:
    (10 + (35000 / процент* корень (удалённость галактики* 20000000 / скорость))) / (24 * 60 * 60)

Потребление дейтерия:
  • В системе:
    =округл.(кол-во кораблей* потребление корабля* ((1000000 + 5000 * удалённость планеты) / 35000000) * (процент + 1)^2 ;0) +1
    Пример:
    • 1 БТ с 3:22:1 на 3:22:2 со 00%
    • Кол-во кораблей = 1
    • Потребление корабля = 50
    • Удалённость--> |1-2| =1
    • Процент= 100% = 1

    =округл. [ 1 * 50 * ((1000000 + 5000 * 1) / 35000000) * (1 + 1) ^ 2;0] +1
    =округл. [ 50 * 0,0287142 * 4] +1
    =округл. [5,74284] +1
    = 6 + 1
    = 7

  • В системе:
    =округл. ( кол-во кораблей* потребление* ((2700000 + 95000 * удалённость системы) / 35000000) * (процент+ 1) ^ 2 ;0) + 1

  • Между галактиками:
    =округл. ( кол-во кораблей* потребление* ((20000000 * удалённость галактики) / 35000000) * (процент + 1) ^ 2 ;0) + 1
    Для удалённости надо умножать.


К меню
Скорость полёта:
10% за уровень реактивного двигателя (для транспортов, лёгких истребителей, переработчиков, зондов).
20% за уровень импульсного двигателя (большой истребитель, крейсер, колонизатор, бомбардировщик).
30% за уровень гиперпространственного двигателя (линкоры, уничтожители, ЗС).
Обратите внимание, что сокращение касается только базовой скорости. Большой транспорт с теоретическим сжиганием 0 (т.е. невозможно) 1:07:42, со сжиганием 10 (т.е. вдвое быстрее) 0:47:55, а не 0:33:51. Таким образом, увеличение скорости имеется, но не в такой степени, как это может показаться на первый взгляд.
У малого транспорта при исследовании импульсного двигателя 5-го уровня последний становится стандартным двигателем, изменяя базовую скорость. То же самое касается бомбардировщика при исследовании гиперпространственного двигателя 8-го уровня.

Радиус действия ракет:
= (уровень импульсного двигателя * 2) - 1

Время полёта ракет:
= 30 сек. основная скорость + 1 система за минуту
